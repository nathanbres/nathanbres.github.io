Methodology

The model predicts a player's OPS for the following season using five engineered features. Only players with 300+ plate appearances are included. The model is trained on 2022–2024 data and tested on 2025 results.

Target Variable

Next-season OPS (On-base Plus Slugging) — the primary measure of overall offensive production.

Key Features

Current OPS, 2-season rolling average OPS_avg2, next-season age, WAR, and plate appearances PA.

Model

Ordinary Least Squares linear regression with an 80/20 train-test split. Evaluated on RMSE and R².

Data

Baseball Reference standard batting stats, seasons 2022–2025. Minimum 300 PA per season.

Model Limitations & Next Steps

This is a baseline model. Understanding its limits is as important as its predictions.

Low R² (0.209)

The model explains ~21% of variance. Professional models (ZiPS, Steamer) reach 40–50% using many more features.

Small Sample

Only 134 training samples after requiring 3+ seasons of data. More historical data would improve reliability.

No Statcast

Exit velocity, launch angle, and barrel rate are strong predictors not yet included in this model.

No Park Factor

Ballpark effects significantly impact OPS. A Coors Field hitter vs. a pitcher's park hitter need adjustment.

Next Steps

Add Statcast features, park factors, platoon splits, and test Ridge/Lasso regularization to reduce overfitting.
