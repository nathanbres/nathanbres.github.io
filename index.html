Web App · React / Vite / JavaScript / Supabase

The Lineup

A free, mobile-first MLB draft and team simulation web app built solo from scratch. Users draft a 26-man roster from the entire history of MLB, simulate a full season via a PA outcome model calibrated on real OBP/SLG/AVG distributions, and compete on a global leaderboard.

  • Draft engine with position-aware logic and 700+ real MLB players from the Lahman database
  • Simulation model based on empirical plate appearance outcomes (OBP, SLG, AVG)
  • Multilingual support — English, French, and Spanish
  • Supabase leaderboard, Formspree contact form, Google Analytics integration
  • Personal Hall of Fame, player anecdotes, and share functionality
Stack: React, Vite, JavaScript, Supabase Year: 2024–2025
